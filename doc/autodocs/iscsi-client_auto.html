Configuration of iscsi-client

 clients/iscsi-client_auto.ycp
Client for autoinstallation
  • Michal Zugec <mzugec@suse.cz>

This module has an unstable interface.
This is a client for autoinstallation. It takes its arguments, goes through the configuration and return the setting. Does not do any changes to the configuration.

Imports

  • IscsiClient
  • Progress
  • Progress

Includes

  • iscsi-client/wizards.ycp

Info:

Parameters:
function to execute
map /list of iscsi-client settings
Return value:
edited settings, Summary or boolean on success depending on called function
Example 
 map mm = $[ "FAIL_DELAY" : "77" ];
 map ret = WFM::CallFunction ("iscsi-client_auto", [ "Summary", mm ]);
